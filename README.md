# Wise.com Clone

A comprehensive Wise.com clone built with React, TypeScript, Tailwind CSS, and Vite.

## Pages

- **Landing Page** (`/`) — Hero with live currency converter, stats, features, comparison table, testimonials, CTA
- **Login Page** (`/login`) — Split-panel design with email/password form and social login
- **Sign Up Page** (`/signup`) — 3-step flow: Account → Personal details → OTP verification
- **Dashboard** (`/dashboard`) — Sidebar nav, multi-currency balances, transfer chart, transaction history

## Tech Stack

- React 19 + TypeScript
- Tailwind CSS 4
- Vite 7
- React Router v7
- Recharts (area chart)
- Motion (animations)
- Lucide React (icons)
- shadcn/ui components
- Convex (backend/database)

## Color Scheme

- Primary green: `#9FE870`
- Dark green: `#163300`

## Getting Started

```bash
npm install
npm run dev
```
